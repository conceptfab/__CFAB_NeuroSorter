Biblioteka pyqtgraph, której używamy do rysowania wykresów, akceptuje kolory w kilku formatach:
Nazwy kolorów: Możesz używać standardowych nazw kolorów zdefiniowanych w HTML/CSS (i rozpoznawanych przez Qt). Przykłady:
red
blue
green
yellow
black
white
cyan
magenta
gray
orange
purple
itd. (pełna lista jest dość długa, ale te są najczęstsze)
Skrócone nazwy jednoliterowe (dla podstawowych kolorów):
b: niebieski (blue)
g: zielony (green)
r: czerwony (red)
c: cyjan (cyan)
m: magenta
y: żółty (yellow)
k: czarny (black)
w: biały (white)
Wartości heksadecymalne (Hex Color Codes): Tak jak w HTML/CSS, możesz zdefiniować kolor za pomocą jego wartości szesnastkowej, poprzedzonej znakiem #.
#FF0000 (czerwony)
#00FF00 (zielony)
#0000FF (niebieski)
#FFFF00 (żółty)
#000000 (czarny)
#FFFFFF (biały)
#A0A0A4 (odcień szarości, używany np. w pyqtgraph.mkPen)
#3F6E98 (jakiś odcień niebieskiego)
Krotka lub lista wartości (R, G, B) lub (R, G, B, A): Możesz zdefiniować kolor jako krotkę (tuple) lub listę liczb całkowitych z zakresu 0-255, reprezentujących składowe Czerwoną (R), Zieloną (G), Niebieską (B) oraz opcjonalnie Alpha (A - przezroczystość, gdzie 255 to pełna nieprzezroczystość, a 0 to pełna przezroczystość).
(255, 0, 0) (czerwony)
(0, 255, 0, 128) (półprzezroczysty zielony)
[100, 100, 100] (szary)
